# Context: alembic/

## Purpose & Responsibility
Manages database migrations for the SQLite database. It uses the `sqlalchemy.url` defined in `app.core.config` to connect to the database.

## Key Symbols
- `alembic/env.py`:
    - `run_migrations_online()`: The primary entry point for running migrations. It creates an engine from the configuration and runs the migrations within a transaction.
    - `target_metadata`: References `Base.metadata` from `app.backend.database.base` to enable autogenerate support.
- `alembic/script.py.mako`:
    - The Mako template used to generate new migration scripts.

## Sub-directories
- `versions/`: Contains the individual migration scripts.

## Dependencies
- `alembic`: Migration framework.
- `app.core.config`: For `DATABASE_URL`.
- `app.backend.database.base`: For model metadata.
