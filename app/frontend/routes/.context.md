# Context: app/frontend/routes/

## Purpose & Responsibility
Handles the rendering of HTML pages and partials. Manages the high-level browsing logic and server-side state for the UI.

## Key Symbols
- `app/frontend/routes/frontend_routes.py`:
    - `read_path()`: The main browsing route. Handles both full page loads and HTMX partial swaps based on headers.
    - `login_page()`: Renders the login UI.
    - `get_server_stats()`: API endpoint for the dashboard/stats view.

## Dependencies
- `fastapi.templating.Jinja2Templates`: For HTML rendering.
- `app.core.file_security`: For secure path validation during browsing.
