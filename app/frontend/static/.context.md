# Context: app/frontend/static/

## Purpose & Responsibility
Stores all client-side static assets. Organized into CSS, JavaScript, and Icon directories.

## Sub-directories
- `css/`:
    - `style.css`: Custom overrides, glassmorphism effects, and mobile optimizations.
    - `bundle.css`: Aggregated CSS for performance.
    - `artplayer.css`: Styles for the video player.
- `js/`:
    - `main.js`: Global initializations and HTMX event listeners.
    - `sw.js`: Service Worker for media caching and Range request interception.
    - `theme.js`: Dark/Light theme management.
    - `artplayer.js`, `htmx.min.js`, `alpine.min.js`: Vendor libraries.
- `icons/`: SVG icons for file types and UI elements.

## Key Files
- `manifest.json`: PWA manifest for mobile installation.

## Dependencies
- `Service Worker API`: Used in `sw.js` for "zero-loading" video streaming.
