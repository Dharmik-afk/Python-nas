# Context: app/frontend/static/js/

## Purpose & Responsibility
Handles client-side interactivity and background tasks.

## Key Symbols
- `main.js`:
    - HTMX `htmx:beforeSwap` listener to allow 404 swaps for empty folder states.
- `sw.js` (Service Worker):
    - `handleRangeRequest()`: Intercepts Range headers to serve cached media slices, enabling offline-like performance for video seeking.
    - `handleRegularRequest()`: Caches media and thumbnails.

## Dependencies
- `HTMX`: For server communication.
- `Alpine.js`: Used in templates for UI state.
