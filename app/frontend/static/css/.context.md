# Context: app/frontend/static/css/

## Purpose & Responsibility
Manages the visual presentation of the application. Focuses on a modern "glassmorphism" aesthetic and mobile-first responsiveness.

## Key Files
- `style.css`: Contains:
    - **Breadcrumb Scroll:** Horizontal scrolling for long paths on mobile.
    - **Glassmorphism:** `.glass-card`, `.btn-glass`, and `.lightbox-overlay` styles.
    - **Mobile Optimizations:** Specific touch target sizes (44x44px+) and responsive padding.
    - **Lightbox Layout:** Flexbox-based layout for the media viewer.

## Dependencies
- `bootstrap`: Utility classes used extensively.
