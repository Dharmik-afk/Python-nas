# Context: app/frontend/templates/

## Purpose & Responsibility
Contains the Jinja2 templates for the application. Uses a hierarchical structure of layouts, pages, and partials.

## Sub-directories
- `layouts/`: Base structure (HTML head, navbar, footer).
- `pages/`: Full page definitions (file browser, login).
- `partials/`: Reusable components and HTMX fragments.

## Dependencies
- `Jinja2`: Template engine.
- `app.core.templates`: Shared instance and context processors.
