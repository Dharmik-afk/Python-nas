# Context: app/backend/

## Purpose & Responsibility
Contains the backend logic of the application, organized into database management, data models, API routes, and service integrations. It acts as the bridge between the FastAPI frontend and the internal Copyparty file engine.

## Sub-directories
- `database/`: SQLALchemy setup, models, and CRUD operations.
- `models/`: Pydantic schemas for data validation and API responses.
- `routes/`: FastAPI routers for API, Auth, and File operations.
- `services/`: Integrations with external or internal services (e.g., Copyparty).

## Dependencies
- `sqlalchemy`: Database ORM.
- `pydantic`: Data validation.
- `requests`: HTTP client for internal proxying.
