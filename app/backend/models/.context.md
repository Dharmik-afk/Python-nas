# Context: app/backend/models/

## Purpose & Responsibility
Defines Pydantic schemas (Data Transfer Objects) for consistent data exchange between the backend, the mobile API, and the frontend.

## Key Symbols
- `app/backend/models/fs_schemas.py`:
    - `FSItem`: Represents a single file or directory in the filesystem.
    - `DirectoryListing`: Represents a list of items and the directory's permissions.
- `app/backend/models/user_schemas.py`:
    - `UserCreate`, `UserUpdate`, `User`: Schemas for user registration and API responses.
    - `Token`, `TokenData`: Schemas for OAuth2 JWT authentication.

## Dependencies
- `pydantic`: Schema definition and validation.
