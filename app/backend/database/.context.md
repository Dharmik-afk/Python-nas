# Context: app/backend/database/

## Purpose & Responsibility
Manages the SQLite database using SQLAlchemy. Handles schema definition, connection pooling, and basic CRUD operations for user management.

## Key Symbols
- `app/backend/database/base.py`:
    - `Base`: The declarative base for all ORM models.
- `app/backend/database/models.py`:
    - `User`: The user model, storing usernames, hashed passwords (bcrypt for DB, SHA-512 for Copyparty), and permissions.
- `app/backend/database/session.py`:
    - `SessionLocal`: Factory for creating database sessions.
    - `engine`: The SQLAlchemy engine instance.
- `app/backend/database/crud.py`:
    - `get_user()`, `get_user_by_username()`, `create_user()`: Standard CRUD functions for user persistence.

## Dependencies
- `sqlalchemy`: ORM and database engine.
- `app.core.config`: For `DATABASE_URL`.
