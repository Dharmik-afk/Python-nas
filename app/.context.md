# Context: app/

## Purpose & Responsibility
This is the main application package, containing the core FastAPI application and its primary sub-modules (core, backend, frontend, tests).

## Key Symbols
- `app/main.py`:
    - `app`: The FastAPI application instance.
    - `session_middleware`: Middleware for session management and activity logging.
    - `startup_event()`/`shutdown_event()`: Lifecyle hooks for logging and initialization.
    - `get_service_worker()`/`get_manifest()`: Serves PWA-related static files from the root scope.

## Dependencies
- `fastapi`: Web framework.
- `app.core`: Core utilities and configuration.
- `app.backend.routes`: Backend API and file handling routes.
- `app.frontend.routes`: Frontend page routes.
